## Overview
This project analyzes Amazon's top-selling books using Python and Jupyter Notebook. The analysis includes insights into book categories, ratings, pricing, and trends. It is designed to demonstrate data cleaning, visualization, and exploratory data analysis (EDA) skills.

---

## Project Structure

---

## Features
- **Data Cleaning**: Handling missing values, formatting columns, and preparing the dataset for analysis.  
- **Exploratory Data Analysis (EDA)**: 
  - Most popular book categories  
  - Rating distribution analysis  
  - Price trends and patterns  
- **Visualizations**: Interactive charts and graphs using Python libraries such as `matplotlib` and `seaborn`.  
- **Insights**: Provides actionable insights about best-selling books on Amazon.

---

## Technologies Used
- Python 3.x  
- Jupyter Notebook  
- Pandas, NumPy  
- Matplotlib, Seaborn
